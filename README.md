# Confluence CLI

Command line interface for the [Atlassian Confluence REST API](https://developer.atlassian.com/cloud/confluence/rest/v2/intro/).

## Features

- Page
  - Create page
  - Get page by ID
  - Get pages
  - Delete page
  - Update page
- Space
  - Get spaces
